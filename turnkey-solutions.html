Starting Something™

All the boring technical stuff™ before launching a product

It should not be opinionated about development tools, languages, frameworks and runtimes that I can use

However, there are several solutions that simplify the boring technical stuff™ that I want, but are opinionated about development tools, languages and/or runtimes.

If that works for you, great! Here are some I found:

Hosting providers

Some will provide all the boring technical stuff™ pretty easy to set up and turnkey ready.

  • Vercel
  • Cloudflare

Others can supply all the bits and peaces, but you still have to integrate everything yourself.

  • Amazon Web Services (AWS)
  • Microsoft Azure
  • Google Cloud Provider (GCP)

Next.js

  • Shipfast is a payed solution that focuses on adding many of the needed features one-by-one into your next.js based applications. (Show HN: A NextJS boilerplate to automate all the boring stuff.)
  • Next Inject is also a payed solution similar to shipfast. (Show HN: I created a Next.js CLI to ship code like a hacker.)
